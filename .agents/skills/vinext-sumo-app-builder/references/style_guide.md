# Style Guide for o-sumo

大相撲情報サイトのモダンなUIを実現するためのスタイルガイドです。

## カラーパレット
モダンなグラデーションと清潔感のある配色を採用しています。

- **プライマリカラー**: `#667eea` (明るい紫)
- **セカンダリカラー**: `#764ba2` (濃い紫)
- **アクセントカラー**: `#d4af37` (金 - 横綱用)
- **アクセントカラー**: `#c0c0c0` (銀 - 大関用)
- **アクセントカラー**: `#cd7f32` (銅 - 関脇用)
- **背景グラデーション**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`

## レスポンシブデザイン
モバイルファーストの設計を推奨します。

- **デスクトップ**: 1200px以上
- **タブレット**: 768px～1199px
- **モバイル**: 480px～767px
- **小型モバイル**: 479px以下

## コンポーネントスタイル
- **BanzukeTable**: 東西の力士を左右に並べ、番付をバッジで表示。
- **RikishiName**: フォントサイズ1.1rem、ウェイト700を推奨。
- **RikishiYomi**: フォントサイズ0.85rem、イタリック、色#666を推奨。
- **RankBadge**: 角丸20px、太字、階級に応じた背景色を使用。
