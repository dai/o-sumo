# 「場所を掘る」トップページカード設計

## 目的

トップページのHero直下に、大相撲アナリティクスを新コンテンツ「場所を掘る」として強く訴求する。場所中に最優先される取組速報の視認性を維持しながら、勝ち星や決まり手から今場所を深く見る入口を同じ画面内に設ける。

## レイアウト

- Hero直下に `速報` と `場所を掘る` の2カードを横並びで配置する。
- 860px以下では `速報`、`場所を掘る` の順で縦積みにする。
- 既存Hero内の「アナリティクス」テキストリンクは削除し、分析導線を新カードへ一本化する。
- News、決まり手、過去場所など後続セクションの順序は変更しない。

## カード内容

- 見出し: `場所を掘る`
- 補足: `大相撲アナリティクス`
- 説明: `勝ち星、無敗力士、決まり手から、今場所の流れを読み解きます。`
- CTA: `アナリティクスを見る`
- 遷移先: `/analytics/`

## 視覚表現

- 速報カードは既存の墨色と朱色CTAを維持する。
- 分析カードは既存の紙・墨・金トークンを使い、速報カードと役割を区別する。
- 角丸、汎用アイコン、装飾目的のアニメーションは追加しない。
- カードの境界は色面と余白で示し、既存のEditorial Homeデザインに合わせる。
- ライト・ダーク双方で本文4.5:1、見出しと大きなCTA文字3:1以上のコントラストを維持する。

## アクセシビリティと検証

- 2カードを意味のあるセクションとしてマークアップし、見出しで識別できるようにする。
- キーボードフォーカスは既存CTAと同じ明確な表示を使う。
- テストでカード文言、`/analytics/` 導線、Hero→カード群→NewsのDOM順序を固定する。
- Playwrightで1440px、390px、ライト、ダーク、横方向overflowを確認する。
