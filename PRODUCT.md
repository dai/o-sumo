# Product

## Register

product

## Users

日本語話者と英語話者の大相撲ファン。公式（日本相撲協会 Ajax エンドポイント / 公式サイト）よりも速く、構造化され、モバイルでオフラインでも読める番付・星取表・取組結果・予定を欲している層。本場所（年に 6 回、各 15 日間）の進行中、雷雨で電波が弱い本場所会場でも結果を確認できる体験を求める。

主なペルソナ:

- **常連ファン**: 毎場所チェックし、力士の昇進や星取を追いかけている。力士名からプロフィールへ瞬時に飛べること、昇順／降順を切り替えられることを期待する。
- **ライト層**: 結果だけを確認したい。トップに最新の幕内優勝と三賞が並んでいれば十分。
- **海外在住のファン / 翻訳中の学習者**: 日本語表記・読み（yomi）・英語表記の三言語的構造が必要。`rikishi-202605.txt` のような辞書データでカバー。
- **開発者 / データ消費者**: 静的にホストされた JSON API（`/api/v1/*.json`）を読み取ってアプリや bot を作りたい層。

## Product Purpose

大相撲の最新番付と日別取組（結果・予定）を、構造化データとして高速に配信する静的 Web アプリ。Cloudflare Pages からホストされ、静的 JSON API も同時に公開する。`python scripts/update_sumo_data.py` による日次・リアルタイム更新、`/202605-banduke` `/20260524-yotei` のような日付ベース URL、PWA によるオフライン対応によって、本場所 15 日間を滞りなく追えるようにすることが成功の尺度。

## Brand Personality

**静謐（serene）・伝統（traditional）・緊張（discipline）**

"The Digital Washi"（DESIGN.md）と "The Digital Dojo"（DESIGN-Ronin.md）の二系統を擁する。Zen の *Ma*（余白）の哲学で画面を組み立て、声は編集者的で威厳がある。色と素材のパレットは Ink / Paper / Gold（明系統）または Midnight / Neon（Ronin 暗系統）に固定し、装飾よりもタイポグラフィと余白で重みを示す。

## Anti-references

- **アニメ・萌え・派手色**: 二次創作的・キャラクター的な色遣いや装飾、和の静けさを壊す強い彩度、過剰なエフェクト、汎用的なグラデーション。
- **SaaS デフォルト**: Inter / 角丸 12px / Indigo グラデーション / ヒーローメトリック / 同一カードグリッドの繰り返し。
- **AI デフォルトの暖色オフホワイト + tracked eyebrow + 番号付きセクション + グラデーション h1**: 2026 年時点で AI が出力しがちな画面構成（cream body / `ABOUT 01 / PROCESS 02 / PRICING 03` の eyebrow / グラデ見出し）のいずれかを含めない。

## Design Principles

1. **Ma 優先**: 画面が詰まったら要素を減らす。狭めるのではなく省く。余白は省略記号として扱う。
2. **墨と紙と金**: パレットは Ink / Paper / Gold（または Midnight / Neon）に固定。色を増やすのではなくレイヤーの濃淡で立体感を出す。
3. **エディトリアル・タイポグラフィ**: Noto Serif JP と Space Grotesk のテンションで紙面の重みを示す。グラデーションで誤魔化さない。
4. **オフライン・ファースト**: Service Worker が日次データをキャッシュし、電波が届かない会場でも結果が確認できるようにする。
5. **データ可視性を最優先**: 力士名・番付・星取が主役。装飾はそれが読まれる限りにおいてのみ許される。
6. **本場所運用の確実性**: 15 日分の URL を先出しし、未更新日は `pending` プレースホルダーで空状態を見せる。データ欠損で 404 を返さない。

## Accessibility & Inclusion

WCAG 2.1 AA を目標とする。フォーカスリング・`sr-only`・`prefers-reduced-motion` のサポートを既定で組み込む。日本語（`lang="ja"`）と英語（`lang="en"`）の二言語を提供し、`<html lang>` と各ページの言語属性を一致させる。配色は Light / Dark（Ronin）の二系統で、いずれも本文テキストのコントラスト比 4.5:1 以上を維持する。
